# KSN 智能汇总单生成器

AI 驱动的 PDF 排柜图解析工具，自动从 PDF 排柜图中提取托盘、套装、包装清单数据，生成结构化 JSON 和 Excel 汇总表。支持多用户并发使用。

## 部署文件清单

只需以下文件即可运行：

```
项目根目录/
├── 物料数据库.db
│
└── 汇总-拆分/
    ├── 启动.bat              ← 双击即可运行
    ├── server.py
    ├── AI物流表.html
    ├── 托盘展开版本_en.py
    ├── 托盘展开版本_ja.py
    ├── input/
    │   └── 提示词_1.docx
    └── output/
        ├── logo_en.png
        └── logo_ja.png
```

## 安装与启动

**方式一：一键启动（推荐）**

1. 确保已安装 [Python 3.10+](https://www.python.org/downloads/)（安装时勾选 "Add Python to PATH"）
2. 双击 `启动.bat`
3. 首次运行自动安装依赖，之后直接启动服务
4. 浏览器打开 `http://127.0.0.1:5000`

**方式二：手动启动**

```bash
pip install flask flask-cors requests pypdfium2 pdfplumber openpyxl
cd 汇总-拆分
python server.py
```

## 使用流程

1. 打开页面 → 展开顶部「API 配置」填写模型和 Key
2. 填写物料参数（托盘数量、英文套装、中文清单）
3. 上传 PDF 排柜图
4. 发送模式下拉框选择：**仅文本**（默认）/ **图片+文本** / **仅图片**
5. 点击「AI 自动提取编排」，AI 流式输出解析结果
6. 右侧聊天面板实时显示 AI 回复，可随时点击「停止」中断
7. 解析完成后自动展开 JSON 面板，可点击「编辑」在线修改，修改后点「保存」
8. 需要补充缺失页面时：在聊天框输入要求 → 点「补充页面」→ 输入页码按回车，自动合并发送
9. 点击「生成英文Excel」或「生成日文Excel」生成汇总表
10. 下载 Excel 或 JSON

## 界面布局

页面采用左右分栏布局：
- **左侧（420px）**：配置面板 — API 设置、物料参数、PDF 上传、发送模式、操作按钮
- **右侧（自适应）**：对话面板 — 聊天记录、JSON 数据手风琴、Excel 下载

## 功能说明

### AI 对话
- 聊天框输入消息与 AI 对话，AI 会参考已有 JSON 数据上下文
- 回复超长时可点「继续输出」，自动带上文继续生成
- 每次对话累计统计 token 用量和预估费用

### JSON 数据合并
- 三组 JSON 数据（托盘清单、英文套装、中文清单）各自独立管理
- 多次提取时按关键字段去重合并，新数据追加不覆盖
- 手风琴折叠面板支持展开查看、编辑、保存

### 补充页面
- 先在聊天框输入补充要求（可选），再点「补充页面」输入页码
- 系统自动提取 PDF 指定页面文本，与聊天框文字合并发送给 AI
- 提取结果与已有 JSON 合并，不丢失之前的数据

### Excel 生成
- 文件名自动包含订单号和日期（如 `38062+38625_0612_en.xlsx`）
- 支持英文版和日文版，分别由 `托盘展开版本_en.py` 和 `托盘展开版本_ja.py` 生成

## 多用户支持

- 每次点击「AI 自动提取编排」创建全新会话，状态完全重置
- 文件存储在 `sessions/<sessionId>/` 下，互不干扰
- 闲置超过 1 小时的会话自动清理
- 同一局域网内多人可同时访问 `http://<服务器IP>:5000`

## 修改提示词

编辑 `input/提示词_1.docx`，保存后下次请求自动生效，无需重启。

## 依赖

- Python 3.10+
- flask, flask-cors, requests, pypdfium2, pdfplumber, openpyxl
